# ✨ Telegram Fresh

> 🚀 Современная локальная реализация **tg-ws-proxy** для Telegram Desktop с использованием WebSocket.

**Telegram Fresh** — лёгкое приложение, вдохновлённое идеей **tg-ws-proxy**, которое организует локальный мост между Telegram Desktop и сетью Telegram через WebSocket.

Проект ориентирован на максимально простую настройку, стабильную работу и минимальное потребление ресурсов. Все соединения обрабатываются локально, а необходимые параметры формируются автоматически без необходимости поднимать собственную серверную инфраструктуру.

> ⚡ Использование **telegram-ws-proxy** позволяет во многих случаях уменьшить задержки подключения, ускорить синхронизацию чатов, загрузку медиа и передачу файлов.

---

## 🌟 Основные возможности

- 🚀 Современная реализация **tg-ws-proxy**
- 🌐 Передача трафика через WebSocket
- 🔒 Полностью локальная обработка соединений
- ⚡ Быстрый запуск без сложной настройки
- 🔄 Автоматическое переподключение при сбоях
- 📄 Подробное журналирование
- 💻 Поддержка Windows, Linux и macOS
- 🪶 Минимальная нагрузка на систему
